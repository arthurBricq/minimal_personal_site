More about how this page was built

This is my attempt to write an extremely lightweight, yet pleasant, personal website. It uses only the basics: HTML, CSS and JS without any dependency. I use to have a personal website (which I was very happy about) written with the React framework gatsby. After not touching the code for 1 year and a half, it happened to have 195 broken dependencies. Just that. This experience made me mad about the current state of these newly shiny JS frameworks, and I ended up writing a personal website without any JS dependency. All of the text is written in markdown files, that are parsed in python, and then exported in html through a home-made website generator.
